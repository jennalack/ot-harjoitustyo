# LintuApp

Sovelluksen avulla käyttäjien on mahdollista pitää kirjaa näkemistään lintulajeista. Sovellusta voi käyttää useampi rekisteröitynyt käyttäjä, joilla jokaisella on oma listansa lintuhavainnoistaan.
